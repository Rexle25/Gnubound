OpenGeoPuzzle

Das ist das Github repo: https://github.com/Rexle25/OpenGeoPuzzle

Mit dieser Seite kannst du komplett kostenlos (Auch keine Werbung. Diese Seite bleibt für immer kostenlos) virtuelle Schnitzeljagden erstellen.

Du kannst simpel und benutzerfreundlich eine Schnitzeljagd erstellen. Diese kannst du dann als JSON exportieren.

Du kannst einfach das JSON einfügen und spielen. Der Spielstand wird automatisch im browser gespeichert.

Ich arbeite momentan noch daran mehr features hinzuzufügen.
