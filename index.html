OpenGeoPuzzle

GitHub Repository Play Create

This is the GitHub repo: https://github.com/Rexle25/OpenGeoPuzzle

With this site you can create virtual treasure hunts completely free of charge (no ads either. This site will remain free forever).

You can create a treasure hunt simply and user-friendly. You can then export it as JSON.

You can simply paste the JSON and play. Progress is saved automatically in the browser.

I am currently working on adding more features.

Have fun playing and creating!
